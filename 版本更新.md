## v1.0
- 基础版本

## v1.1
- 修复 Module View 中叶子节点全部显示 "Code" 无法辨识的问题，现在显示具体模块名
- 修复 Webview 加载后 treemap 区域空白的问题（JS 语法错误）
- Module View 中 depth=2 的模块节点增加分组标题显示
- 优化容器高度计算逻辑，增加 fallback 机制

## v1.2
- 支持配置 ROM 和 RAM 大小，覆盖 MAP 文件中的 Max 值
- 按文件名独立存储配置，不同 MAP 文件互不影响
- 工具栏添加齿轮按钮，点击弹出输入框设置（单位 KB，0 表示使用 MAP 文件原始值）
- 配置持久化到 .vscode/settings.json

## v1.3
- 支持点击查看详细参数功能，例如AK620-GEN2.map的digital.show.o，不一下子显示所有函数的信息，在点开后，显示新的视图，里面有所有的函数

## v1.4

- 直接把模块名名放到模块中显示，例如digital.show.o显示在模块的中间，字体的大小根据模块的大小决定
- back按键放到ROM前面

## v1.5
